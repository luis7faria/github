Como funciona?

Este software baixa videos em massa a partir dos ids do youtube em 480p, o código pode ser rescrito sempre mantendo autoria do programa.

Este programa/software é baseado em uma aplicação que web que permite baixar um video por vez no navegador, similar ao keepvid/clipconverter entre outros, a versão original está disponível em: https://github.com/jeckman/YouTube-Downloader/ (licensa oficial na pasta li/)

O algumas partes do código original deste programa foram alteradas, e foram adicionados também arquivos para fazer o processamento e o download, para tornar o processamento mais rápido foi usado iframes uma vez que o php não suporta multi-thread na sua forma original, os iframes criam um processo por video o que acelera bastante o processo.

O código deste programa não está comentado, contudo se for necessário poderá estar nas próximas versões, para que o programa funcione corretamente verifique as suas configurações do php.ini, em alguns casos as configurações padrão tem bloqueios de limite de memória, verifique as configurações do seu php, em caso de alguma duvida você tem o meu contato. :. Sugestões são bem vindas .:
